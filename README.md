Usage:

```html
<clock-face 
time-zone="America/Toronto"
second-hand="true"
></clock-face>
```

CSS Variables:

```css
      clock-face{
        --clock-size:100%;
        --clock-face-fill:none;
        --clock-face-outline:currentColor;
        --clock-hand-outline: currentColor;
        --clock-accent-color: orange; /* used for second hand */ 
        --clock-stroke-width: 1.5;
      }
```
